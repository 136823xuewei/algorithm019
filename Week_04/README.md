**二分查找**
1、使用二分查找，寻找一个半有序数组中间无序的地方：  
  **特点：所有变化点左侧元素>数组第一个元素**  
  **所有变化点右侧元素<数组第一个元素**  
  **算法**  
  1、找到数组中间mid  
  2、比较nums[mid]与nums[0]大小，若nums[mid]>nums[0],在mid右边搜索，否则在mid左边搜索  
  3、找到变化点时停止搜索，以下条件满足任意一个即可  
    nums[mid]>nums[mid+1]，因此mid+1是最小值  
    nums[mid-1]>nums[mid]，因此mid是最小值  
    最小值处是变化点
